# HMSS Error Codes

| Code | Description |
|------|-------------|
| 0 | Unknown error |
| 1 | Permission denied |
| 2 | Root user is protected |
| 3 | Not found (user, media, resource, etc.) |
| 4 | Invalid or missing input |
| 5 | Missing dependency or module |
| 6 | Required configuration missing |
| 7 | No changes to apply |
| 8 | Reserved value (e.g. username "root") |
| 9 | Database operation failed |
| 10 | Invalid credentials |
| 11 | Username already exists |
| 12 | Session expired |
| 13 | Account is disabled |
| 20 | Media not found |
| 21 | Unsupported media format |
| 22 | Transcoding unavailable |
| 23 | Storage is full |
| 30 | Stream not available |
| 31 | Stream limit reached |
| 32 | Bandwidth exceeded |
| 40 | Addon not found |
| 41 | Addon load failed |
| 42 | Addon incompatible |
| 50 | Internal server error |
